Assignment Objective

This assignment is designed to simulate a real-world data visualization task where you will analyze, deconstruct, and accurately recreate a provided visualization using a tool of your choice (you can use more if you like). One of the most important skills in the workforce is choosing the right tool for the job, getting a sense for varying levels of features, interactivity, and customizability available with different visualization software tools, and figuring out which tools you would rather avoid (this will help you when it comes time to pick tools for your group project!).

Write some code to recreate the visualization(s) as accurately as you can using one or more visualization tools from the list of approved tools. Document your process.
